# ✏️ **Nano 편집기 실습 예제 (Beginner Practice)**

---

## 🧩 1️⃣ 실습 목적

* `nano` 편집기의 기본 조작법 익히기
* 파일 생성, 저장, 편집, 검색, 복사/붙여넣기, 줄 이동, 종료까지 실습
* 터미널 기반 문서 편집 환경에 익숙해지기

---

## 🧠 2️⃣ 사전 준비

터미널(또는 WSL, Ubuntu shell 등) 실행 후, 연습용 폴더를 만듭니다.

```bash
mkdir nano_practice
cd nano_practice
```

---

## 🧾 3️⃣ 실습 ① — 새 파일 만들기

아래 명령으로 `memo.txt` 파일을 만듭니다.

```bash
nano memo.txt
```

이제 빈 화면이 열립니다.
아래 내용을 그대로 입력해보세요 👇

```
오늘의 할 일
1. Python 복습
2. Pixi 환경 실습
3. Nano 편집기 연습
```

---

## 💾 4️⃣ 실습 ② — 저장하고 나가기

1. `Ctrl + O` → Enter (저장)
2. `Ctrl + X` (종료)

그럼 터미널로 돌아옵니다.
파일이 저장되었는지 확인해보세요 👇

```bash
cat memo.txt
```

---

## 🧮 5️⃣ 실습 ③ — 내용 추가하기

다시 파일을 열어보세요:

```bash
nano memo.txt
```

맨 아래 줄로 내려가서 다음 줄을 추가합니다.

```
4. 새로운 Python 프로젝트 시작
```

저장 후 종료:
→ `Ctrl + O` → Enter → `Ctrl + X`

---

## 🔍 6️⃣ 실습 ④ — 텍스트 검색하기

파일을 다시 열고:

```bash
nano memo.txt
```

`Ctrl + W` 를 누르고
검색어로 **Python** 을 입력한 뒤 **Enter** 를 눌러보세요.
👉 커서가 “Python”이 들어간 줄로 자동 이동됩니다.

---

## ✂️ 7️⃣ 실습 ⑤ — 잘라내기 & 붙여넣기

1. **커서를 “Pixi 환경 실습” 줄로 이동**
2. `Ctrl + K` → 잘라내기
3. **맨 아래로 이동** (`Ctrl + _` → 줄 번호 입력 or `Ctrl + V`)
4. `Ctrl + U` → 붙여넣기

이제 목록 순서가 바뀌었습니다.

---

## 🪄 8️⃣ 실습 ⑥ — 줄 복사하기 (Alt + 6)

1. 커서를 “Python 복습” 줄로 이동
2. `Alt + 6` → 복사
3. 커서를 맨 아래로 이동
4. `Ctrl + U` → 붙여넣기

👉 줄 복사와 붙여넣기가 완성됩니다.

---

## 🧭 9️⃣ 실습 ⑦ — 문단 정렬 (Ctrl + J)

아래 내용을 추가해보세요:

```
이 문장은 길게 써서 정렬 실습을 해보기 위한 예제 문장입니다 nano의 문단 정렬 기능을 이용하면 문장이 자동으로 줄 바꿈되어 보기 좋게 바뀝니다.
```

`Ctrl + J` 를 눌러보세요.
→ 긴 문장이 자동으로 줄이 정리됩니다!

---

## 🧰 10️⃣ 실습 ⑧ — 현재 위치 확인 (Ctrl + C)

아무 줄에서나 `Ctrl + C`를 눌러보세요.
→ 아래쪽 상태 표시줄에 **줄(Line) / 열(Column)** 번호가 표시됩니다.

---

## 🧨 11️⃣ 실습 ⑨ — 다른 이름으로 저장

저장할 때 이름을 바꾸려면:
`Ctrl + O` → `Ctrl + T` → 새 이름 입력 (`backup.txt`) → Enter
→ `Ctrl + X` 로 종료

이제 파일이 두 개 생겼습니다.

```bash
ls
```

```
memo.txt  backup.txt
```

---

## 🧹 12️⃣ 실습 ⑩ — 파일 삭제 후 복원하기

```bash
rm memo.txt
cp backup.txt memo.txt
```

→ 이렇게 하면 백업 파일을 다시 원본 이름으로 복원할 수 있습니다.

---

## ✅ 13️⃣ 실습 요약

| 기능       | 단축키        | 설명              |
| -------- | ---------- | --------------- |
| 저장       | `Ctrl + O` | 파일 저장           |
| 종료       | `Ctrl + X` | nano 종료         |
| 잘라내기     | `Ctrl + K` | 현재 줄 잘라내기       |
| 붙여넣기     | `Ctrl + U` | 붙여넣기            |
| 복사       | `Alt + 6`  | 현재 줄 복사         |
| 검색       | `Ctrl + W` | 문자열 찾기          |
| 줄 이동     | `Ctrl + _` | 특정 줄로 이동        |
| 문단 정렬    | `Ctrl + J` | 자동 줄바꿈 정렬       |
| 커서 위치 표시 | `Ctrl + C` | 현재 줄/열 번호 보기    |
| 도움말      | `Ctrl + G` | nano 단축키 도움말 보기 |
