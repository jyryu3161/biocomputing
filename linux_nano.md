# 🧾 **리눅스 Nano 편집기 기초**

## 1️⃣ Nano란?

* `nano`는 리눅스/유닉스 환경에서 가장 간단하고 직관적인 **터미널 기반 텍스트 편집기**입니다.
* `vi`나 `vim`보다 사용법이 단순하여, 초보자들이 설정 파일 수정이나 간단한 스크립트 편집 시 많이 사용합니다.

---

## 2️⃣ Nano 실행 및 종료

| 명령어                | 설명                            |
| ------------------ | ----------------------------- |
| `nano`             | 새 빈 문서 열기                     |
| `nano filename`    | 파일을 열거나, 없으면 새로 생성            |
| `nano +3 filename` | 파일을 열고 **3번째 줄**로 바로 이동       |
| `Ctrl + X`         | 편집 종료 (Exit)                  |
| `Y` / `N`          | 종료 시 변경사항 저장 여부 묻는 질문에 Yes/No |

예시:

```bash
nano myscript.sh
```

→ 편집 후 `Ctrl + O` 로 저장, `Ctrl + X` 로 종료

---

## 3️⃣ Nano 기본 편집 명령

| 명령어        | 설명                   |
| ---------- | -------------------- |
| `Ctrl + O` | 파일 저장 (Write Out)    |
| `Ctrl + X` | 종료 (Exit)            |
| `Ctrl + G` | 도움말 보기 (Help)        |
| `Ctrl + J` | 문단 정렬 (Justify)      |
| `Ctrl + K` | 현재 줄 잘라내기 (Cut line) |
| `Ctrl + U` | 붙여넣기 (Uncut / Paste) |
| `Ctrl + ^` | 블록 선택 시작             |
| `Alt + 6`  | 선택 영역 복사             |
| `Ctrl + C` | 현재 커서 위치(줄/열 번호) 표시  |

---

## 4️⃣ 이동 관련 단축키

| 단축키              | 설명                 |
| ---------------- | ------------------ |
| `Ctrl + A`       | 현재 줄의 **맨 앞으로 이동** |
| `Ctrl + E`       | 현재 줄의 **맨 끝으로 이동** |
| `Ctrl + Y`       | **이전 페이지**로 이동     |
| `Ctrl + V`       | **다음 페이지**로 이동     |
| `Ctrl + _` (언더바) | **특정 줄 번호로 이동**    |
| `Alt + /`        | 파일의 맨 끝으로 이동       |
| `Ctrl + /`       | 파일의 맨 앞으로 이동       |

---

## 5️⃣ 검색 및 바꾸기

| 명령어        | 설명                            |
| ---------- | ----------------------------- |
| `Ctrl + W` | 문자열 검색 (Where Is)             |
| `Alt + W`  | 같은 검색 반복 (다음 찾기)              |
| `Ctrl + \` | 문자열 바꾸기 (Search and Replace)  |
| `Ctrl + T` | 맞춤법 검사기 (spell checker, 설치 시) |

예시:

```
Ctrl + W → 검색어 입력 → Enter
Ctrl + \ → 바꾸기 대상 및 새 문자열 입력
```

---

## 6️⃣ 파일 관련 고급 기능

| 명령어        | 설명                        |
| ---------- | ------------------------- |
| `Ctrl + R` | 다른 파일 내용 불러오기 (Read File) |
| `Ctrl + O` | 현재 파일 저장 (Write Out)      |
| `Alt + F`  | 현재 파일 이름 변경 후 저장          |
| `Alt + A`  | 선택 영역 설정 (Mark Text)      |
| `Alt + T`  | 파일 내 전체 텍스트 잘라내기          |

---

## 7️⃣ 설정 및 환경

* Nano의 설정 파일은 **`~/.nanorc`** 에 위치합니다.
  이 파일에서 색상 강조, 자동 들여쓰기 등 옵션을 지정할 수 있습니다.

예시 (`~/.nanorc`):

```bash
set autoindent
set linenumbers
set tabsize 4
set softwrap
```

* **라인 번호 표시**: `Ctrl + C`로 확인 가능하거나, `.nanorc`에서 `set linenumbers` 설정

## 🧭 9️⃣ Nano 단축키 요약표

| 분류    | 단축키                     | 기능          |
| ----- | ----------------------- | ----------- |
| 저장/종료 | `Ctrl + O`              | 저장          |
|       | `Ctrl + X`              | 종료          |
|       | `Ctrl + G`              | 도움말         |
| 편집    | `Ctrl + K`              | 잘라내기        |
|       | `Ctrl + U`              | 붙여넣기        |
|       | `Alt + 6`               | 복사          |
| 이동    | `Ctrl + A` / `Ctrl + E` | 줄의 처음 / 끝   |
|       | `Ctrl + Y` / `Ctrl + V` | 이전 / 다음 페이지 |
| 검색    | `Ctrl + W`              | 찾기          |
|       | `Ctrl + \`              | 바꾸기         |
| 표시    | `Ctrl + C`              | 커서 위치 보기    |
| 기타    | `Ctrl + _`              | 특정 줄 이동     |
|       | `Alt + F`               | 다른 이름으로 저장  |
